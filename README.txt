# -Library-Maps
data structure, homework 7
A program used to maitain the holding and checkout information for a library.
The input .txt file will give the instruction for the program to add a book, checkout a book .etc.
For example:
a 2 pride and prejudice: This operation will add the specified number of copies of the named 
book or dvd to the library.
The .pdf file will give more information about this simple program:
